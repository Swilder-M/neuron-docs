# EtherNet/IP(CIP)

## Parameter Configuration

| Field | description                |
| ----- | -------------------------- |
| host  | device ip                  |
| port  | device port, default 44818 |
| slot  | cpu slot, default 0        |

## Support Data Type

* INT8
* UINT8
* INT16
* UINT16
* INT32
* UINT32
* INT64
* UINT64
* FLOAT
* DOUBLE
* BOOL
* BIT
* STRING
* WORD
* DWORD
* LWORD
